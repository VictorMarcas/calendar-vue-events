# Calendar Events

## Plugins
- Vue 2.6
- Tailwindcss 3.0
- v-calendar [https://vcalendar.io/](https://vcalendar.io/)

## Start project
``` bash
$ npm install
```

## Mode Development
config file `webpack.dev.js`
``` bash
$ npm run dev
```
## Mode Production 
config file `webpack.prod.js`

``` bash
$ npm run prod
```

### Production ready build.
- JavaScript minified
- HTML minified
- CSS minified
